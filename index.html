Lekcja 4.05 | Jednostki

Jednostki miary:

  • Piksele |px| // podstawowa jednostka miary
  • Procenty |%| // jednostka przyjmuje procentową wartość elementu nadrzędnego
  • Vievport width |vw| // element przyjmuje szerokość okna przeglądarki nie patrząc na elementy nadrzędne
  • Vievport height |vh| // element przyjmuje wysokość okna przeglądarki nie patrząc na elementy nadrzędne

Lekcja 4.04 | Życie w pudełku

Display i overflow:

  • display block // element blokowy domyślnie jeden w lini, może przybierać wartości wymiarów
  • display inline // elementy przybierają wymiary kontentu i ustawiają się obok siebie
  • display inline-block // elementy ustawiają się obok siebie oraz mogą przyjmować wartości z display block
  • overflow hidden // ukrywa kontet wychodzący poza wymiary obiektu nadrzędnego
  • overflow visible // pokazuje kontent nawet gdy wychodzi poza wymiary elementu nadrzednego
  • overflow auto // pozwala na przewijanie nadwyżki kontentu który znajduje się w elemencie nadrzędnym

Lekcja 4.02 | Git komendy

Lista przydatnych komend do gita:

  • git config --global user.name "Nazwa uzytkownika" //nadaja nazwe dla usera
  • git config --global user.email adres@email.com //dodaje email do usera
  • git init //towrzy repozytorium
  • git status //podgląd repozytorium
  • git add . //dodaj / odświerza nowe pliki
  • git commit -m "nazwa" //robi commit
  • gitk //otiwera okno repozytorium
  • git push //wysyłamy commit na serwer
